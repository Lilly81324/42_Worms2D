# Chat Contracts

This folder contains chat package examples currently used in the app:

- **Client package examples**: payloads sent by frontend clients to BFF endpoints.
- **Server package examples**: payloads sent from BFF to the internal social service (`/internal/...`).

These examples are derived from current routes in:

- `apps/BFF/src/modules/social/social.controller.ts`
- `apps/social/src/modules/social/social.controller.ts`
- `apps/social/src/modules/social/social.dto.ts`
